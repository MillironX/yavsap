Private API Reference

Describes the input/output of every workflow, process, and function within the pipeline.

Workflows

PHYLOGENETIC_TREE

Create a phylogenetic tree

Input
Tuple
prefix(val(String))

Sample identifier
Alignment(file)

Plain-text multi-alignment file. RAxML-NG supports FASTA, PHYLIP, and CATG formats

Output
Tuple
val(String)

Sample identifier
path

Annotated support tree

QC

Perform context-sensitive QC on fastq reads

READS_INGEST

Take reads from the input folder or a samplesheet and reformat them to be single files with clean names

Output
Tuple
val(String)

Sample identifier
path

Reads files

Call haplotypes for Illumina reads using CliqueSNV

Tuple

Unique identifier for this sample
bamfile(file)

Alignment file to call variants from
Output
Tuple
val(String)

The sample identifier passed through `sampleName`
path

The sequences of the found haplotypes in FASTA format
Tuple
val(String)

The sample identifier passed through `sampleName`
path

Descriptions of the found haplotypes in JSON format

HAPLINK_FASTA

Convert a YAML describing haplotypes into a fasta file with the sequences of those haplotypes

Input
Tuple
prefix(val(String))

The identifier for this sample as used in the output filename
haplotypes(file)

The YAML file describing the SNPs in each haplotype
reference(file)

Reference genome to mutate sequences in to create the haplotype sequences
Output
Tuple
val(String)

The identifier as passed though `prefix`
path

The mutated sequences in fasta format

HAPLINK_HAPLOTYPES

Call haplotypes from long reads using HapLink.jl

Input
Tuple
prefix(val(String))

The identifier for this sample as used in the output filename
bamfile(file)

File for the alignment to call haplotypes from
variants(file)

File containing variants to consider haplotypes make of
Output
Tuple
val(String)

The identifier as passed though `prefix`
path

The found haplotypes described in YAML format

HAPLINK_VARIANTS

Call variants for Oxford Nanopore reads using HapLink.jl

Input
Tuple
prefix(val(String))

The identifier for this sample as used in the output filename
bamfile(file)

File for the alignment to call variants from
reference(file)

Reference genome to call variants from in fasta format
Output
Tuple
val(String)

The identifier as passed though `prefix`
path

The variants found in the reads in VCF format

HAPLOTYPE_DEPTH

Convert the `frequency` key in a HapLink.jl output file to a valid (integer) read depth

Input
Tuple
prefix(val(String))

The identifier for this sample
haplotypes(file)

A HapLink.jl file containing a single haplotype definition. If the file contains more than one haplotype, then the process will only return results for the first haplotype.

Output
Tuple
val(String)

The identifier as passed through `prefix`
env

The read depth of the haplotype as an integer

Simulate Illumina paired-end reads using Kraken2’s perl script

Input
Tuple

The unique id for this mutation pattern
genome(file)

Fasta file containing the sequence to simulate reads of
depth(val(Int))

The number of reads to simulate
Output
path

The fastq files containing the simulated reads

Downloads a genome in fasta and genbank format from the NCBI databases

The requested genome in GenBank format

reverseReads(file)

Collection of individual reverse reads files
Output

val(String)

path
