YAVSAP (Yet Another Viral Subspecies Analysis Pipeline)
=======================================================

YAVSAP is a Nextflow pipeline to analyze viral haplotypes and populations within
a single sample, with a focus on Japanese Encephalitis Virus (JEV). We're still
waiting to be inspired with a better name.
